Brows

This page is a demo for this "browser" made by me.

It uses curl to download web pages and raylib to render content.

Controls

You have probably figured out you can scroll pages using your mouse. The 'j' and 'k' keys also scroll down and up.
The 'l' key enters a "debug mode" and shows how the browser calculates position of all the elements. Press 'h' to exit that mode.
Up and down arrows zoom in and out, 't' and 'b' keys scroll to the top and bottom. The 'r' key reloads the page.

Why does this even exist?

Almost everything that is done on a computer can be done in a web browser. This simple implementation has about 1% functionality of a modern web browser, it lacks JavaScript support, proper CSS support and ignores 99% of the w3 spec. Regardless, this project helped me get a basic understanding of how modern browsers work, and the insanely forgiving nature of HTML and CSS. There is not much practical use this browser has, except seeing how sites break in amusing ways or trying to write websites that render properly within these constraints.
